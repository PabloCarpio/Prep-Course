FOR => Es un blucl de javascript que nos sirve para repetir instrucciones siempre y cuando sepamos la cantida de veces que se va a ejecutar.
&& || ! => Los operadores &&(and, Y) sirven par verificar la veracidad de dos o mas operandos, siempre da true si los dos o mas tambien son true. El operador ||(or, O) sirve para verificar la veracidad de uno entre varios operandos, basta que uno sea true para que todo sea true, si todos son false da como respuesta false. El operador !(NOT) sirve para negar cualquier valor booleano,si es true lo devuelve false y viseversa.
